1. start project in local enviroment

```
$ git clone https://github.com/peterroe/ungeui.git
$ yarn
$ yarn dev
```

2. Develop your components in `src/xxx `

3. Add your components info `src/index.ts`

4. write demos in `src/demo/xxx.vue`
