# Transit🔢

一个神奇的组件，让数字也有过渡变化！

## 代码演示

```js
import { UTransit } from 'ungeui';
```

### 基本用法📦️

注意，组件内部会使用`Number()`对子元素转换，所以该组件的直接子元素必须是数字⚠️

## Props

| 参数  |   说明   |   类型    |   默认值    |
| :---: | :------: | :-------: | :---------: |
| duration | 过渡持续时间，单位ms ｜ `number` |  `500`|
