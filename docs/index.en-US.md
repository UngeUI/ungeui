## Unge UI

Ungeui is a vue3 component library

## install

```shell
yarn add ungeui
```

## use

```js
import { createApp } from 'vue';
import UngeUI from 'ungeui';
import 'ungeui/dist/style.css';

const app = createApp(App);
app.use(UngeUI);
```
